透過以下API，可以下一個任務
http://127.0.0.1:6600/set_mission

任務裡面可包含很多動作，動作可以重復組合
例如:

"task_arr":[ ["MoveTo",[-19.1,28.48,-3.14,"1F"]],  
             ["MoveTo",["N100","1F"]],
             ["FollowRail", ["rail_No.345", 0, 700, "1F"] ],
             ["ButtonWait", 0  ]             
           ] 
           
這樣一個任務裡面就有4個動作

"robot_mode":[0,0,0],                 # 格式為list，0 => off
                                      #             1 => on
                                      # [上UV燈、下UV燈、吸塵器]

機器額外設備設備開關是跟著任務走，而不是動作
